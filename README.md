# Information

### Options

```
    --help          Muestra ayuda                                        [booleano]
    --version       Muestra número de versión                            [booleano]
    -b, --base      Number to multiply                         [número] [requerido]
    -l, --list      Shows the multiplication table                 [defecto: false]
     u, --until     Indicates how far to multiply            [número] [defecto: 10]
```
